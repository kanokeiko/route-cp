# Google Maps Route Copier

A simple Chrome extension to copy Google Maps transit route details as plain text.

## What it does
- Extracts the currently visible directions/route details in Google Maps
- Shows them in the popup
- One-click copy to clipboard

## Notes
- Works only on https://www.google.com/maps/
- May break if Google changes their UI
